#How to run migration
 - Open package menager console -> View -> Other Windows -> Package manager console
 - Chose the default project with the applicationDbContext (Notes.Infrastracure)
 - run `update-database`

#How to add migration
 - Open package menager console -> View -> Other Windows -> Package manager console
 - Chose the default project with the applicationDbContext (Notes.Infrastracure)
 - run `add migration [Name]`
